## Installation

1. Clone the repository:

```bash
git clone https://github.com/diwashbhattarai999/Gov-Certify.git

```

2. Install dependencies:

```bash
pnpm install
```

3. Generate prisma:

```bash
npx prisma generate
```


4. Start the server:

```bash
# if running dev server
pnpm dev

# if running prod server
pnpm build
pnpm start
```
